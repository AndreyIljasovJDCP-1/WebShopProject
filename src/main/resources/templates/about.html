Проект Web-Shop на основе данного видео-курса

В этом проекте реализована имитация интернет-магазина, на основе Spring Boot с возможностью хранения, добавления, удаления товаров, а также авторизации и регистрации пользователей с иерархией ролей и привилегий, с возможностью их редактирования пользователем с правами Администратора. При разработке блока Spring Security использовались материалы этой статьи.
